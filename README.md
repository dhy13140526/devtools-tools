# devtools-tools

Free online developer tools for JSON formatting, Base64 encoding, UUID generation, timestamp conversion, URL encoding, and more.

## Features

- **UUID Generator**: Generate unique UUIDs (version 4)
- **Base64 Encoder/Decoder**: Encode or decode text using Base64 encoding
- **Timestamp Converter**: Convert between Unix timestamps and human-readable dates
- **URL Encoder/Decoder**: Encode or decode URLs to safely transmit special characters
- **JSON Formatter/Validator**: Format, validate, and beautify JSON data

All tools run in your browser for maximum privacy. No server-side processing required.

## Usage

Visit [https://pcgithub.io/devtools-tools](https://pcgithub.io/devtools-tools) to start using the tools.

## Tools

### UUID Generator
Generate unique UUIDs (version 4) for testing, development, or database keys.

### Base64 Encoder/Decoder
Encode or decode text using Base64 encoding for data transmission and storage.

### Timestamp Converter
Convert between Unix timestamps and human-readable dates, and vice versa.

### URL Encoder/Decoder
Encode or decode URLs to safely transmit special characters and reserved characters.

### JSON Formatter/Validator
Format, validate, and beautify JSON data with syntax highlighting and error checking.

## Privacy

All tools run locally in your browser. No data is sent to any server, ensuring maximum privacy and security.
